1.  Подключите систему M-MDS к диагностическому разъему DLC-2.

am3zzw00000466

2.  После идентификации автомобиля выберите последовательно следующие пункты меню на экране M-MDS.

•  При использовании IDS (портативного ПК)
1.  Выберите “Module Programming”.
•  При использовании PDS (карманного ПК)
1.  Выберите “Programming”.
2.  Выберите “Module Programming”.

3.  Затем выберите в меню на экране “Programmable Module Installation” и “PCM”.

4.  Выберите “Programmable Parameters”. [с DSC HU/CM]

5.  Выберите “Tire Size / Axle Ratio”. [с DSC HU/CM]

6.  Выберите “Body Type - (PCM)”, после чего на экране M-MDS выберите тип кузова. [с DSC HU/CM]

7.  При помощи диагностической системы M-MDS проверьте наличие в памяти кодов неисправностей.

•  Если какой-либо диагностический код присутствует, выполните соответствующую этому коду процедуру проверки.
