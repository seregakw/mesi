TIMING BELT REMOVAL/INSTALLATION [MZ-CD 1.6 (Y6)]

id0110b2804000

1.  Remove the battery cover.

2.  Disconnect the negative battery cable.

3.  Remove the following parts:

(1)  Engine cover
(2)  Drive belt (See DRIVE BELT REPLACEMENT [MZ-CD 1.6 (Y6)].)
(3)  Under cover and splash shield
(4)  Coolant reserve tank with the hoses and pipes still connected (See СНЯТИЕ И УСТАНОВКА РАСШИРИТЕЛЬНОГО БАЧКА [MZ-CD 1.6 (Y6)].)

4.  Position the fuel main and return hoses out of the way.

5.  Remove the wiring harness.

6.  Remove in the order indicated in the table.

7.  Install in the reverse order of removal.

8.  Start the engine.

9.  Inspect the pulley and belt for runout and contact and adjust them if necessary.

Upper timing belt cover

No.3 Engine Mount Rubber Removal Note

1.  Secure the engine and the transaxle using an engine jack and attachment.

am3zzw00001363


Timing Belt Pulley, Timing Belt Removal Note

1.  Align the camshaft pulley using the SST.

am3zzw00001364

2.  Loosen the timing belt tensioner installation bolt.

am3zzw00001365

3.  Remove the timing belt pulley.

Осторожно
•  Do not touch the outer sensor ring.
am3zzw00001366


Timing Belt Pulley, Timing Belt Installation Note

1.  Align the fuel pump pulley using the SST.

am3zzw00004597

2.  Remove the SST the fuel pump pulley.

3.  Install the timing belt pulley.

4.  Install the crankshaft installation bolt

5.  Align the crankshaft using the SST.

am3zzw00001368

6.  Align the camshaft pulley using the SST.

am3zzw00001369

7.  Remove the SST from the camshaft pulley.

8.  Install the timing belt.

9.  Tension the timing belt using the following steps.

Примечание
•  Using an hexagonal wrench, maintain tension until the timing belt tensioner bolt is tightened.
(1)  Turn the timing belt tensioner adjustment arm counterclockwise to apply slight tension to the timing belt.
am3zzw00001370
(2)  The pointer must be positioned between the sides of the window.
(3)  Tighten the timing belt tensioner installation bolt.

10.  Remove the SST.

am3zzw00001368

11.  Rotate the crankshaft 12 revolutions.

Осторожно
•  Only rotate the crankshaft in the normal direction of rotation.

12.  Install the SST.

am3zzw00001368

•  If the SST cannot be installed, repeat the previous steps.

13.  Remove the SST.

am3zzw00001368

14.  Install the SST.

am3zzw00001364

•  If the SST cannot be installed, repeat the previous steps.

15.  Remove the SST.

am3zzw00001364

16.  Install the SST.

am3zzw00004598

•  If the SST cannot be installed, repeat the previous steps.

17.  Remove the SST.

am3zzw00004598


Crankshaft Pulley Installation Bolt Installation Note

1.  Lock the flywheel using the SST.

am3zzw00001362

•  Rotate the crankshaft until the special tool aligns with one of the slots in the flywheel.

2.   Tighten the crankshaft pulley installation bolt in two steps.

1.  Tightening to 30 N·m {31 kgf·m, 22 ft·lbf}
2.  Tightening to 180°
