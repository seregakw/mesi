DRIVE BELT REPLACEMENT [MZ-CD 1.6 (Y6)]

id0110b2803800

1.  Remove the engine cover.

2.  Rotate the drive belt auto tensioner clockwise using the SST as shown in the figure until the pin hole on the auto tensioner body appears.

d3e110cw4001

Примечание
•  Make sure the SST is securely set so that the SST will not be slip off the of the auto tensioner projection during securing process.

3.  Insert the SST into the pin hole on the auto tensioner body.

d3e110cw4003

4.  Remove the drive belt.

5.  Install the new drive belt.

6.  Remove the SST.

7.  Install the engine cover.
