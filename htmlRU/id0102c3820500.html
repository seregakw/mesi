DTC P0402

EGR flow excessive detected
DETECTION CONDITION
•  The PCM monitors the MAF sensor signal voltage while EGR system is operating. If the actual MAF signal is below desired signal in PCM, the PCM determines that there is a malfunction in the EGR system.
POSSIBLE CAUSE
•  EGR valve malfunction
•  Air suction at intake-air system
•  Intake throttle valve stuck close
•  PCM malfunction
