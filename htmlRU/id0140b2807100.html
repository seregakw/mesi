FUEL ADDITIVE CONTROL MODULE REMOVAL/INSTALLATION [MZ-CD 1.6 (Y6)]

id0140b2807100

Примечание
•  If a new fuel additive control module is being installed connect the M-MDS. Upload the fuel additive control module configuration information using the programable modules installation routine prior to starting the removal of the fuel additive control module. (See FUEL ADDITIVE CONTROL MODULE CONFIGURATION [MZ-CD 1.6 (Y6)].)
Perform the “FUEL ADDITIVE LEVEL DATA RESET” after performing the “FUEL ADDITIVE CONTROL MODULE CONFIGURATION”. (See FUEL ADDITIVE LEVEL DATA RESET [MZ-CD 1.6 (Y6)].)

1.  Remove the battery cover. (See СНЯТИЕ И УСТАНОВКА АККУМУЛЯТОРНОЙ БАТАРЕИ [MZ-CD 1.6 (Y6)].)

2.  Disconnect the negative battery cable.

3.  Remove the rear seat cushion.

4.  Remove the fuel additive control module connector.

5.  Remove the fuel additive control module installation nuts.

6.  Remove the fuel additive control module.

am3zzw00001120

7.  Install in the reverse order of removal.

Tightening torque
2.5—3.5 N·m {25.5—35.6 kgf·cm, 22.2—30.9 in·lbf}
