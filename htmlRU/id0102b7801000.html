Процедура чтения кодов неисправностей

1.  Подключите диагностическую систему M-MDS к диагностическому разъему DLC-2.

am3uuw00000151

2.  После идентификации автомобиля последовательно выберите перечисленные ниже позиции меню на стартовом экране M-MDS.

•  При использовании IDS (портативного ПК)
1.  Выберите кнопку “Toolbox”.
2.  Выберите “Self Test”.
3.  Выберите “Modules” (электронные модули).
4.  Выберите “PCM”.
•  При использовании PDS (карманного ПК)
1.  Выберите “Module Tests”.
2.  Выберите “PCM”.
3.  Выберите “Self Test”.

3.  Затем, выберите “Retrieve CMDTCs” (вывести сохранённые коды неисправностей) и выполняйте указания, выводимые на экран диагностической системы M-MDS.

4.  Проверяйте полученные коды в соответствии с указаниями на экране диагностической системы M-MDS.

•  Для показанных диагностических кодов выполните соответствующую процедуру поиска и устранения неисправности.

5.  После устранения неисправностей удалите все коды неисправностей, сохранённые в блоке РСМ, действуя во соответствии с указаниями раздела “ПОСЛЕРЕМОНТНЫЕ МЕРОПРИЯТИЯ“.

Процедура доступа к проверке готовности системы бортовой диагностики

1.  Подключите диагностическую систему M-MDS к диагностическому разъему DLC-2.

am3uuw00000151

2.  После идентификации автомобиля последовательно выберите перечисленные ниже позиции меню на стартовом экране M-MDS.

•  При использовании IDS (портативного ПК)
1.  Выберите кнопку “Toolbox”.
2.  Выберите опцию “Powertrain”.
3.  Выберите “OBD Test Modes”.
4.  Выберите “Mode 1 Powertrain Data” (режим 1 параметры трансмиссии).
•  При использовании PDS (карманного ПК)
1.  Выберите “OBD II Modes”.
2.  Выберите “Mode 1 Powertrain Data” (режим 1 параметры трансмиссии).

3.  Затем выберите параметры “***SUP” и “***EVAL” на экране выбора параметров (PID).

4.  Проверьте значения выбранных параметров и убедитесь в завершении проверки системы.


Монитор параметров и данных (PID/DATA MONITOR) и процедура записи

Примечание
•  Функция PID/DATA MONITOR позволяет наблюдать за расчётными значениями входных и выходных сигналов блока РСМ. Поэтому, сбой значения наблюдаемого параметра выводящего устройства не является непосредственным указанием на неисправность выводящего устройства. Если значение наблюдаемого параметра выводящего устройства не соответствует техническим условиям, то нужно проверить значение наблюдаемого параметра устройства ввода, входящего в данный контур управления.

1.  Подключите диагностическую систему M-MDS к диагностическому разъему DLC-2.

am3uuw00000151

2.  После идентификации автомобиля последовательно выберите перечисленные ниже позиции меню на стартовом экране M-MDS.

•  При использовании IDS (портативного ПК)
1.  Выберите кнопку “Toolbox”.
2.  Выберите “DataLogger”.
3.  Выберите “Modules”.
4.  Выберите “PCM”.
•  При использовании PDS (карманного ПК)
1.  Выберите “Module Tests”.
2.  Выберите “PCM”.
3.  Выберите “DataLogger”.

3.  Выберите параметр (PID) из таблицы параметров.

4.  Проверьте значения параметров в соответствии с выводимыми на экран указаниями.

Активный тест

1.  Подключите диагностическую систему M-MDS к диагностическому разъему DLC-2.

am3uuw00000151

2.  После идентификации автомобиля последовательно выберите перечисленные ниже позиции меню на стартовом экране M-MDS.

•  При использовании IDS (портативного ПК)

2.  Выберите “DataLogger”.
3.  Выберите “Modules”.
4.  Выберите “PCM”.
•  При использовании PDS (карманного ПК)
1.  Выберите “Module Tests”.
2.  Выберите “PCM”.
3.  Выберите “DataLogger”.

3.  Выберите имитируемые параметры из таблицы параметров (PID).

4.  Выполните активный тест и проверьте работу всех компонентов.

•  Если во время проведения активного теста не слышен звук работы реле, электродвигателя или электромагнитного клапана, то можно предположить наличие обрыва или короткого замыкания в проводном жгуте, реле, электродвигателе или в обмотке электромагнитного клапана, либо заедание в механических узлах.
