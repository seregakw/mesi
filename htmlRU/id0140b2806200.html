Осторожно
•  Water penetrating the connector will cause sensor malfunction. To prevent this, be careful not to damage the wiring harnesses or the waterproof connector so as to cause water penetration.
Примечание
•  Before performing the following inspection, make sure to follow the procedure as indicated in the troubleshooting flowchart. (See ПОРЯДОК ИСПОЛЬЗОВАНИЯ ДАННОГО РУКОВОДСТВА.)

Visual Inspection

1.  Remove the IAT sensor No.2. (See INTAKE AIR TEMPERATURE (IAT) SENSOR NO.2 REMOVAL/INSTALLATION [MZ-CD 1.6 (Y6)].)

2.  Verify that there are no metal shavings on the sensor.

3.  Visually inspect the IAT sensor No.2 for the following:

•  Damage
•  Cracks
•  Rusted sensor terminal
•  Bent sensor terminal
―  If there is any malfunction, replace the IAT sensor No.2.

4.  Install the IAT sensor No.2. (See INTAKE AIR TEMPERATURE (IAT) SENSOR NO.2 REMOVAL/INSTALLATION [MZ-CD 1.6 (Y6)].)


Resistance Inspection

1.  Disconnect the IAT sensor No.2 connector.

2.  Measure the resistance between IAT sensor No.2 terminals A and B.

•  If it is not as specified, perform the “Circuit Open/Short Inspection”.

Circuit Open/Short Inspection

1.  Disconnect the PCM connector. (See PCM REMOVAL/INSTALLATION [MZ-CD 1.6 (Y6)].)

2.  Inspect the following wiring harness for open or short circuit (continuity check).

am3zzw00001579


Open circuit

•  If there is no continuity, there is an open circuit. Repair or replace the wiring harness.
―  IAT sensor No.2 terminal B and PCM terminal 242
―  IAT sensor No.2 terminal A and PCM terminal 211


Short circuit

•  If there is continuity, there is a short circuit. Repair or replace the wiring harness.
―  IAT sensor No.2 terminal B and power supply
―  IAT sensor No.2 terminal B and body ground
―  IAT sensor No.2 terminal A and power supply
