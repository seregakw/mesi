Описание проверки, выполняемой системой бортовой диагностики (OBD)

•  Проверка, выполняемая системой бортовой диагностики, служит для определения целостности и работоспособности электрогидравлического усилителя рулевого управления (EHPAS). Система бортовой диагностики выводит результаты проверки по специальному запросу.
•  Кроме того, система бортовой диагностики:
―  Выполняет быструю проверку жидкостного насоса электрогидравлического усилителя рулевого управления, которая обычно проводится в начале каждой процедуры диагностики.
―  Обеспечивает сверку данных после ремонтных работ, что позволяет убедиться в отсутствии появления новых неисправностей в ходе ремонта.
•  Проверка, выполняемая системой бортовой диагностики, состоит из двух частей:
―  Считывание/удаление результатов диагностики, мониторинга и записи параметров.

Считывание/удаление результатов диагностики

•  Данная функция обеспечивает считывание или удаление кодов неисправности из памяти блока EHPAS.


мониторинг и запись PID/данных

•  Данная функция обеспечивает доступ к некоторым параметрам, входным сигналам, расчетным величинами и информации о статусе системы.


Процедура считывания диагностических кодов неисправности

1.  Подключите диагностическую систему M-MDS к диагностическому разъему DLC-2.

am3zzw00001707

2.  После идентификации автомобиля выберите последовательно следующие пункты меню на экране M-MDS.

•  При использовании IDS (портативного ПК)
1.  Выберите “Toolbox”.
2.  Выберите “Self Test”.
3.  Выберите “Modules”.
4.  Выберите “EPS”.
•  При использовании PDS (карманного ПК)
1.  Выберите “Module Tests”.
2.  Выберите “EPS”.
3.  Выберите “Self Test”.

3.  Проверьте диагностический код неисправности в соответствии с указаниями, появляющимися на экране.

•  В случае вывода на дисплей какого-либо диагностического кода неисправности выполните процедуру поиска и устранения неисправностей, соответствующую этому диагностическому коду неисправности.

4.  После устранения неисправности удалите все диагностические коды неисправности из памяти блока управления DSC. (См. СИСТЕМА БОРТОВОЙ ДИАГНОСТИКИ [ЭЛЕКТРОГИДРАВЛИЧЕСКИЙ УСИЛИТЕЛЬ РУЛЕВОГО УПРАВЛЕНИЯ (EHPAS)].)


Процедура удаления диагностического кода неисправности

1.  Подключите диагностическую систему M-MDS к диагностическому разъему DLC-2.

am3zzw00001707

2.  После идентификации автомобиля выберите последовательно следующие пункты меню на экране M-MDS.

•  При использовании IDS (портативного ПК)
1.  Выберите “Toolbox”.
2.  Выберите “Self Test”.
3.  Выберите “Modules”.
4.  Выберите “EPS”.
•  При использовании PDS (карманного ПК)
1.  Выберите “Module Tests”.
2.  Выберите “EPS”.
3.  Выберите “Self Test”.

3.  Проверьте диагностический код неисправности в соответствии с указаниями, появляющимися на экране.

4.  Находясь в экране кодов неисправностей, нажмите кнопку удаления (clear), чтобы удалить коды неисправностей.

5.  Убедитесь, что коды неисправностей больше не показываются.


Процедура мониторинга и записи PID/данных

1.  Подключите диагностическую систему M-MDS к диагностическому разъему DLC-2.

am3zzw00001707

2.  После идентификации автомобиля выберите последовательно следующие пункты меню на экране M-MDS.

•  При использовании IDS (портативного ПК)
1.  Выберите “Toolbox”.
2.  Выберите “Data Logger”.
3.  Выберите “Modules”.
4.  Выберите “EPS”.
•  При использовании PDS (карманного ПК)
1.  Выберите “Module Tests”.
2.  Выберите “EPS”.
3.  Выберите “Data Logger”.

3.  Выберите соответствующие параметры (PID) из таблицы параметров.

4.  Выполните проверку значений параметров в соответствии с указаниями, появляющимися на экране.

Примечание
•  Функция наблюдения значений параметров применяется для отслеживания расчетных значений параметров. Поэтому, если расчётное значение наблюдаемого параметра устройства вывода не укладывается в допустимые пределы, требуется проверка наблюдаемых параметров соответствующих исполнительных устройств. Кроме того, поскольку система не воспринимает неисправность устройства вывода, как отклонение наблюдаемого параметра от нормы, нужна отдельная проверка исполнительного устройства.
