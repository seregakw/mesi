MASS AIR FLOW (MAF) SENSOR INSPECTION [MZR-CD (RF Turbo)]

id0140f1800700

Note
•  Perform the following test only when directed.

Voltage Inspection

1.  Turn the engine switch to ON.

2.  Monitor the MAF PID using the M-MDS.

•  If not as specified, perform the “Circuit Open/Short Inspection”.
―  If there is no open/short circuit, replace the MAF/IAT sensor. (See INTAKE-AIR SYSTEM REMOVAL/INSTALLATION [MZR-CD (RF Turbo)].)
MAF PID
0.0—1.0V

3.  Start the engine and warm up the engine completely.

4.  Monitor the MAF PID using the M-MDS at idle.

•  If not as specified, perform the “Circuit Open/Short Inspection”.
―  If there is no open/short circuit, replace the MAF/IAT sensor. (See INTAKE-AIR SYSTEM REMOVAL/INSTALLATION [MZR-CD (RF Turbo)].)
MAF PID
1.5—2.1 V


Circuit Open/Short Inspection

1.  Disconnect the PCM connector. (See PCM REMOVAL/INSTALLATION [MZR-CD (RF Turbo)].)

2.  Inspect for open/short circuit in the following wiring harnesses.

•  If there is open/short circuit, repair or replace wiring harnesses.
am3zzw00004082


Open circuit

•  MAF/IAT sensor terminal A and PCM control relay terminal C
•  MAF/IAT sensor terminal B and PCM terminal 73
•  MAF/IAT sensor terminal C and PCM terminal 54


Short circuit

•  MAF/IAT sensor terminal A and power supply
•  MAF/IAT sensor terminal A and body ground
•  MAF/IAT sensor terminal B and power supply
•  MAF/IAT sensor terminal C and power supply
•  MAF/IAT sensor terminal C and body ground
