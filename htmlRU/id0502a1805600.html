id0502a1805600

1.  Подключите диагностическую систему M-MDS к диагностическому разъему DLC-2.

am3zzw00000458

2.  После идентификации автомобиля выберите следующие пункты меню в окне начального этапа диагностической системы M-MDS.

•  При использовании IDS (портативного ПК)
1.  Выберите “Toolbox”.
2.  Выберите “DataLogger”.
3.  Выберите “Modules”.
4.  Выберите “PCM”.
•  При использовании PDS (карманного ПК)
1.  Выберите “Module Tests”.
2.  Выберите “PCM”.
3.  Выберите “DataLogger”.

3.  Выберите PID из таблицы PID

4.  Убедитесь, что PID соответствует указаниями, появляющимися на экране M-MDS.

Примечание
•  Проведите проверку выводящего устройства после проверки блока РСМ.
•  Параметры, контролируемые в режиме /DATA MONITOR, представляют собой расчетные значения входных/выходных сигналов блока PCM. Поэтому, если контролируемый параметр устройства вывода не соответствует техническим условиям, необходимо проверить контролируемый параметр устройства ввода, соответствующего выходному. Если ошибка выходного параметра не отображается непосредственно как неисправность, необходимо проверить устройство вывода отдельно, используя имитационный тест.

Порядок использования функции симуляции

1.  Подключите диагностическую систему M-MDS к диагностическому разъему DLC-2.

am3zzw00000458

2.  После идентификации автомобиля выберите следующие пункты меню в окне начального этапа диагностической системы M-MDS.

•  При использовании IDS (портативного ПК)
―  Выберите “Toolbox”.
―  Выберите “DataLogger”.
―  Выберите “Modules”.
―  Выберите “PCM”.
•  При использовании PDS (карманного ПК)
―  Выберите “Module Tests”.
―  Выберите “PCM”.
―  Выберите “DataLogger”.

3.  Выберите с помощью системы M-MDS перечисленные параметры для имитации:

4.  Для проверки работы каждого компонента выполните имитационный тест.

•  Если при проведении имитационного теста из электромагнитного клапана не слышны характерные щелчки, свидетельствующие о его работе, значит возможно наличие обрыва или короткого замыкания в цепи электромагнитного клапана или его зависание и выход из строя.
