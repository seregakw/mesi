DTC P1102

Mass air flow (MAF) sensor in range but lower than expected
DETECTION CONDITION
•  The PCM compares actual input from MAF sensor with expected MAF amount that PCM calculates by engine running condition. If the actual MAF signal is excessive lower than expected value, the PCM determines that the MAF sensor signal is lower than expected.
POSSIBLE CAUSE
•  MAF sensor malfunction
•  Electric corrosion in MAF signal circuit
•  Voltage drops in MAF sensor signal circuit
•  PCM malfunction
