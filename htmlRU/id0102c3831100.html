DTC P0339

Crankshaft position (CKP) sensor circuit intermittent concern
DETECTION CONDITION
•  The PCM monitors the input voltage from the CKP sensor signal. If the voltages at PCM deviates from the threshold, the PCM determines that the there is a CKP sensor problem.
POSSIBLE CAUSE
•  CKP sensor circuit intermittent concern
•  CKP sensor pulse wheel malfunction
•  PCM malfunction
