MASS AIR FLOW (MAF) SENSOR INSPECTION [MZ-CD 1.6 (Y6)]

id0140b2800700

Осторожно
•  Water penetrating the connector will cause sensor malfunction. To prevent this, be careful not to damage the wiring harnesses or the waterproof connector so as to cause water penetration.
Примечание
•  Before performing the following inspection, make sure to follow the procedure as indicated in the troubleshooting flowchart. (See ПОРЯДОК ИСПОЛЬЗОВАНИЯ ДАННОГО РУКОВОДСТВА.)

Visual Inspection

1.  Remove the MAF sensor. (See INTAKE-AIR SYSTEM REMOVAL/INSTALLATION [MZ-CD 1.6 (Y6)].)

2.  Verify that there are no metal shavings on the sensor.

3.  Visually inspect the MAF sensor for the following:

•  Damage
•  Cracks
•  Rusted sensor terminal
•  Bent sensor terminal
―  If there is any malfunction, replace the MAF sensor.

4.  Install the MAF sensor. (See INTAKE-AIR SYSTEM REMOVAL/INSTALLATION [MZ-CD 1.6 (Y6)].)


Voltage Inspection

1.  Start the engine and warm up the engine completely.

2.  Measure the voltage between MAF/IAT sensor terminal B and body ground.

Circuit Open/Short Inspection

1.  Disconnect the PCM connector. (See PCM REMOVAL/INSTALLATION [MZ-CD 1.6 (Y6)].)

2.  Disconnect the MAF sensor connector.

3.  Inspect the following wiring harness for an open or short circuit. (Continuity check)

am3zzw00001568
am3zzw00001569

Open circuit

•  If there is no continuity, the circuit is open. Repair or replace the wiring harness.
―  MAF sensor terminal C and PCM control relay terminal E
―  MAF sensor terminal B and PCM terminal 124
―  MAF sensor terminal E and PCM terminal 229


Short circuit

•  If there is continuity, the circuit is shorted. Repair or replace the wiring harness.
―  MAF sensor terminal C and body ground
―  MAF sensor terminal B and power supply
―  MAF sensor terminal B and body ground
―  MAF sensor terminal E and power supply
