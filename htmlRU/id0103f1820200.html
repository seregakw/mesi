DIESEL PARTICULATE FILTER LAMP FLASHES
DESCRIPTION
•  Diesel particulate filter lamp flashes because the diesel particulate filter system can not dispose of diesel particulate
POSSIBLE CAUSE
•  Driving condition is not in diesel particulate filter regeneration condition
•  Diesel particulate filter is not regenerated by manual regeneration procedure
•  Diesel particulate filter damaged or deterioration
•  Erratic signal input to PCM
―  Exhaust gas pressure sensor signal
•  PCM does not realize the diesel particulate trap amount in diesel particulate filter
