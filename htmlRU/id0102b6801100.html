ПОСЛЕРЕМОНТНЫЕ МЕРОПРИЯТИЯ [LF АКП (FN4A-EL), L3 АКП]

1.  Подключите диагностическую систему M-MDS к диагностическому разъему DLC-2.

am3zzw00003527

2.  После идентификации автомобиля последовательно выберите перечисленные ниже пункты меню на стартовом экране диагностической системы M-MDS.

•  При использовании IDS (ноутбука)
1.  Выберите кнопку “Toolbox”.
2.  Выберите “Self Test”.
3.  Выберите “Modules”.
4.  Выберите “PCM”.
5.  Выберите “Retrieve CMDTCs” (вызвать сохранённые коды неисправностей).
•  При использовании PDS (карманного ПК)
1.  Выберите “Module Tests”.
2.  Выберите “PCM”.
3.  Выберите “Self Test”.
4.  Выберите “Retrieve CMDTCs” (вызвать сохранённые коды неисправностей).

3.  Проверяйте полученные коды в соответствии с указаниями на экране диагностической системы M-MDS.

4.  Находясь в экране кодов неисправностей, нажмите кнопку удаления, чтобы удалить коды неисправностей.

5.  Убедитесь, что диагностические коды больше не показываются.
