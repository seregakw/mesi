DTC P2458

Diesel particulate filter regeneration duration
DETECTION CONDITION
•  The PCM monitors the diesel particulate filter regeneration duration using the exhaust differential pressure sensor. If the exhaust differential pressure sensor signal high condition is longer than specified period. The PCM determines that the diesel particulate filter regeneration duration is too long.
POSSIBLE CAUSE
•  EGR valve stuck
•  Air bypass valve stuck
•  Intake throttle valve stuck
•  Exhaust gas leakage in exhaust system before diesel particulate filter
•  Improper connection at exhaust differential pressure sensor upstream side pipe
•  Diesel particulate filter malfunction
•  Exhaust differential pressure sensor malfunction
•  Fuel additive system malfunction
•  Low engine compression
•  No fuel additive in fuel additive tank
•  Scheduled maintenance does not have followed
•  MAF sensor malfunction
•  FACM malfunction
•  PCM malfunction
