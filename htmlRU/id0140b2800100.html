INTAKE THROTTLE VALVE BASE VALUE RESET [MZ-CD 1.6 (Y6)]

id0140b2800100

1.  Connect the M-MDS to DLC-2.

am3zzw00001117

2.  After the vehicle is identified, select the following items from the initial screen of the M-MDS.

Примечание
•  Either procedure A or B can be used.
•  Procedure when using the IDS (notebook PC)
Procedure A
1.  Select “Powertrain”.
2.  Select “Service Function”.
3.  Select “Component/Parameter Reset”.
Procedure B
1.  Select “Programming”.
2.  Select “Module Programming”.
3.  Select “Programmable Parameters”.
4.  Select “Reset EGR Valve Position Sensor Offset”.
•  Procedure when using the PDS (pocket PC)
Procedure A
1.  Select “All Tests and Calibrations”.
2.  Select “Component /Parameter Reset”.
Procedure B
1.  Select “Programming”.
2.  Select “Module Programming”.
3.  Select “Programmable Parameters”.
4.  Select “Reset EGR Valve Position Sensor Offset”.

3.  Then, select “Reset EGR Throttle Valve Position Offset” and perform procedures according to the directions on the M-MDS screen.

4.  Retrieve DTC’s by the M-MDS, then verify that there in no DTC present.

•  If a DTC is present, perform the applicable DTC inspection. (See DTC TABLE [MZ-CD 1.6 (Y6)].)
