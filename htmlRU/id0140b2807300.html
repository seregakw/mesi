Примечание
•  Regarding the Injector Calibration Code, if the PCM is newly replaced, the PCM records a DTC (P1675 and P1676) and illuminates the malfunction indicator lamp even if there is no malfunction. Therefore, if the PCM or fuel injector is replaced, program the injector calibration code in the PCM using the following procedure.
•  Cylinder 1 and fuel injector 1 are located at the end of the transaxle.
am3zzw00001122

1.  Connect the M-MDS to DLC-2.

am3zzw00001121

2.  After the vehicle is identified, select the following items from the initial screen of the M-MDS.

•  When using the IDS (notebook PC)
1.  Select “Module Programming”.
•  When using the PDS (pocket PC)
1.  Select “Programming”.
2.  Select “Module Programming”.

3.  Then, select “Programmable Parameters” and “Injector Correction Factors”, perform procedures according to directions on the M-MDS screen.

4.  Retrieve DTC’s using the M-MDS, then verify that DTCs are present.

•  If a DTC is present, perform the applicable DTC inspection. (See DTC TABLE [MZ-CD 1.6 (Y6)].)
