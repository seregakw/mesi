KOEO/KOER SELF TEST [MZ-CD 1.6 (Y6)]

Примечание
•  Warm up until the exhaust gas temperature (EXHTEMP1, EXHTEMP2, EXHTEMP3 PID) is 100 °C {212°F} or more before performing “KOER SELF TEST”.

1.  Connect the M-MDS to the DLC-2.

am3zzw00000495

2.  Retrieve the KOEO/KOER DTCs using the M-MDS.

3.  After the vehicle is identified, select the following items from the initial screen of the M-MDS.

•  When using the IDS (notebook PC)
1.  Select the “Toolbox” tab.
2.  Select “Self Test”.
3.  Select “Modules”.
4.  Select “PCM”.
•  When using the PDS (pocket PC)
1.  Select “Module Tests”.
2.  Select “PCM”.
3.  Select “Self Test”.

4.  Then, select the “KOEO On Demand Self_Test” or “KOER On Demand Self Test” and perform procedures according to directions on the M-MDS screen.

5.  Verify the DTC according to the directions on the M-MDS screen.

•  If any DTCs are displayed, perform troubleshooting according to the corresponding DTC inspection.

6.  After completion of repairs, clear all DTCs stored in the PCM, while referring to “AFTER REPAIR PROCEDURE”.
