EGR VALVE POSITION SENSOR INSPECTION [MZR-CD (RF Turbo)]

Note
•  Perform the following test only when directed.

Voltage Inspection

1.  Turn the engine switch to ON.

2.  Monitor the EGRP PID using the M-MDS.

•  If not as specified, perform the “Circuit Open/Short Inspection”.
―  If there is no open/short circuit, replace the EGR valve. (See EGR VALVE REMOVAL/INSTALLATION [MZR-CD (RF Turbo)].)
EGRP PID
0.0—1.0 V

3.  Start the engine and warm up the engine completely.

4.  Monitor the EGRP PID using the M-MDS at idle.

•  If not as specified, perform the “Circuit Open/Short Inspection”.
―  If there is no open/short circuit, replace the EGR valve. (See EGR VALVE REMOVAL/INSTALLATION [MZR-CD (RF Turbo)].)
EGRP PID
Approx. 2 V


Circuit Open/Short Inspection

1.  Disconnect the PCM connector. (See PCM REMOVAL/INSTALLATION [MZR-CD (RF Turbo)].)

2.  Disconnect the EGR valve connector.

3.  Inspect for open/short circuit in the following wiring harnesses.

•  If there is open/short circuit, repair or replace wiring harnesses.

am3zzw00004103

Open circuit

•  EGR valve terminal A and PCM terminal 45
•  EGR valve terminal C and PCM terminal 70
•  EGR valve terminal E and PCM terminal 53


Short circuit

•  EGR valve terminal A and power circuit.
•  EGR valve terminal A and body ground.
•  EGR valve terminal C and power circuit.
•  EGR valve terminal E and power circuit.
•  EGR valve terminal E and body ground.
