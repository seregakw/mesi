1.  Connect the M-MDS to the DLC-2.

am3zzw00001928

2.  Verify the following vehicle conditions:

•  Ignition switch is at ON position.
•  All the switches are turned off (except the ignition switch).
•  All the doors, bonnet, and trunk lid or liftgate are closed.
•  All the doors, and trunk lid or liftgate are unlocked.
•  Parking brake lever is pulled.

3.  After the vehicle is identified, select the following items from the initial screen of the M-MDS.

When using the IDS (laptop PC)
•  Select the “Toolbox” tab.
•  Select “Self Test”.
•  Select “Module”.
•  Select “GEM”.
When using the PDS (Pocket PC)
•  Select “Module Tests”.
•  Select “GEM”.
•  Select “Self Test”.

4.  Verify the DTC according to the directions on the screen.

•  If any DTCs are displayed, perform troubleshooting according to the corresponding DTC inspection.

5.  After completion of repairs, clear all DTCs stored in the instrument cluster.
