ENGINE COOLANT TEMPERATURE (ECT) SENSOR REMOVAL/INSTALLATION[ZJ, ZY, Z6]

id0140b1801900

1.  Remove the battery cover. (See BATTERY REMOVAL/INSTALLATION[ZJ, ZY, Z6].)

2.  Disconnect the negative battery cable.

3.  Perform the following procedure for easier access.

Except U.K. specs.
1.  Remove the air cleaner. (See INTAKE-AIR SYSTEM REMOVAL/INSTALLATION[ZJ, ZY, Z6].)
U.K. specs.
1.  Remove the battery and battery tray. (See BATTERY REMOVAL/INSTALLATION[ZJ, ZY, Z6].)
2.  Set the air cleaner cover out of the way. (See INTAKE-AIR SYSTEM REMOVAL/INSTALLATION[ZJ, ZY, Z6].)
3.  Remove the air cleaner case.

4.  Set the EGR pipe (intake manifold side) out of the way for easier access. (See INTAKE-AIR SYSTEM REMOVAL/INSTALLATION[ZJ, ZY, Z6].)

5.  Drain the engine coolant. (See ENGINE COOLANT REPLACEMENT[ZJ, ZY, Z6].)

6.  Install the SST to the ECT sensor.

am3zzw00000467

7.  Remove the ECT sensor.

8.  Replace the gasket.

9.  Install in the reverse order of removal.

ECT sensor tightening torque
15.7—23.5 N·m {1.7—2.3 kgf·m, 12.3—16.6 ft·lbf}
