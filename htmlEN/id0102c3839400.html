DTC P1927

Fuel additive level too low/empty
DETECTION CONDITION
•  The fuel additive control module monitors the fuel additive amount in fuel additive tank using fuel additive level sensor. If fuel additive control module detects that empty fuel additive tank or measured additive value in fuel additive tank is much below reference value, the fuel additive control module determines the fuel additive is too low/empty in fuel additive tank.
POSSIBLE CAUSE
•  No fuel additive in fuel additive tank
•  Fuel additive control module malfunction
