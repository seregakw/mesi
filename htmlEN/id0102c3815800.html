OBD DRIVE MODE[MZ-CD 1.6 (Y6)]

id0102c3815800

Note
•  After a malfunction has been rectified, the malfunction memory must be erased with the aid of the M-MDS. Once the malfunction memory has been cleared, DTC P1000 then remains in the data memory. DTC P1000 is called the readiness code. The readiness code indicates that not all the monitoring systems have completed their tests since the malfunction memory was erased. The readiness code can only be cleared once all the tests by the individual monitoring systems have been successfully completed during a readiness trip. The vehicle must only returned to the customer after the readiness code has been cleared. Only then is it possible to provide assurance that the vehicle is normal.

1.  Performing the Drive Mode inspects the OBD system for proper operation and must be performed to ensure that no additional DTCs are present.

2.  Turn the engine switch to the ON position (Engine off).

3.  Clear all diagnostic data using the M-MDS.

4.  Turn the engine switch off.

5.  Turn the engine switch to the ON position (Engine off).

6.  Verify that only DTC P1000 is present.

•  If any DTC exists, diagnose the applicable DTC inspection.
Caution
•  While performing the Drive Mode, always operate the vehicle in a safe and lawful manner.
•  When the M-MDS is used to observe the monitor system status while driving, be sure to have another technician with you, or record the data in the M-MDS using the PID/DATA MONITOR AND RECORD function, and inspect later.

7.  Drive the vehicle while following the drive mode procedure:

(1)  Start the engine and idle it for more than 1,170 s.
(2)  Turn the engine switch off and wait for more than 15 s.
(3)  Start the engine and idle it for more than 1,170 s.
(4)  Turn the engine switch off and wait for more than 15 s.
(5)  Start the engine and idle it for more than 1,170 s.
(6)  Turn the engine switch off and wait for more than 15 s.

8.  Verify that any DTC has been erased.

•  If DTC P1000 exists, turn the engine switch off then repeat from Step 3.
•  If a DTC except for P1000 exists, diagnose using the applicable DTC inspection.

9.  Turn the engine switch off.

10.  Start the engine.

11.  Verify that no DTCs are present.
