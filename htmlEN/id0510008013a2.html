CLUTCH RELEASE CYLINDER REMOVAL/INSTALLATION[J65M-R]

id0510008013a2

1.  Remove the transaxle.

(See MANUAL TRANSAXLE REMOVAL/INSTALLATION[J65M-R].)

Note
•  Only remove the clutch release cylinder component if it is damaged or leakage.

2.  Remove in the order indicated in the table.

3.  Install in the reverse order of removal.

Note
•  Wrap the primary shaft splines with adhesive tape to protect the primary shaft oil seal.
•  Apply sealer to the mating surfaces of the clutch housing.
•  Install a new clutch release cylinder component.
Caution
•  Do not lubricate the clutch release cylinder.
