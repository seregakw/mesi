OIL CONTROL VALVE (OCV) REMOVAL/INSTALLATION[ZJ, ZY, Z6]

id0110b1801300

1.  Remove the battery cover.

2.  Disconnect the negative battery cable.

3.  Remove the air cleaner component. (See INTAKE-AIR SYSTEM REMOVAL/INSTALLATION[ZJ, ZY, Z6].)

4.  Remove the accelerator cable and bracket. (without throttle valve actuator)

5.  Remove the ground. (No.3 engine mount)

6.  Using the bolts part number 99794 1025 or M10×1.25, length 25 mm {0.98 in} to install the SST to the position shown in the figure.

am3zzw00001861

Caution
•  When attaching the SST in the engine rear side, install a suitable nut between the engine and the SST.
am3zzw00001862
Tightening torque
38—51 N·m {3.9—5.2 kgf·m, 29—37 ft·lbf}

7.  To install the front shaft (RH) of the SST (49 C017 5A0), remove the clip shown the figure.

am3zzw00001863

8.  Install the SST using the following procedure.

Caution
•  Refer to the SST instruction manual for the basic handing procedure.
am3zzw00001864
(1)  Install the right rear shaft of the SST to the bolt of the right shock absorber as shown in the figure.
am3zzw00001865
(2)  Install the left rear shaft of the SST to the bolt of the left shock absorber. (Identical position to the right side)
(3)  Install front foot No.2 to the left/right front shaft of the SST, then align the groove of the front shaft of the SST with the folded up part of the vehicle as shown in the figure.
am3zzw00001866
(4)  Adjust the positions of the SST side bars so that they are the same height (left and right) and horizontal.
(5)  Make sure each joint is securely tightened.

9.  Suspend the engine using the SST.

am3zzw00001867

10.  Remove the No.3 engine mount.

11.  Disconnect the OCV connector.

12.  Remove the OCV installation bolt.

13.  Remove the OCV.

am3zzw00001868

14.  Install the new O-ring.

15.  Install the OCV.

16.  Tighten the OCV installation bolt.

Tightening torque
7.8—10.8 N·m {80—110 kgf·cm, 69.1—95.5 in·lbf}

17.  Connect the OCV connector.

18.  Tighten the No.3 engine mount installation stud bolts.

am3zzw00001869

Tightening torque
7—13 N·m {0.8—1.3 kgf·m, 5.2—9.5 ft·lbf}

19.  Install the No.3 engine mount, and then temporarily tighten the installation bolts and nuts.

20.  Tighten the installation bolts and nuts in the order shown in the figure.

am3zzw00001870

Tightening torque
1 (nuts): 74.5—104.9 N·m {7.60—10.6 kgf·m, 55.0—77.3 ft·lbf}
2 (bolts): 74.5—93.1 N·m {7.60—9.49 kgf·m, 55.0—68.6 ft·lbf}

21.  Install the clip as shown in the figure.

am3zzw00001863

22.  Install the ground. (No.3 engine mount)

23.  Install the accelerator cable and bracket. (without throttle valve actuator)

24.  Install the air cleaner component. (See INTAKE-AIR SYSTEM REMOVAL/INSTALLATION[ZJ, ZY, Z6].)

25.  Connect the negative battery cable.

26.  Install the battery cover.
