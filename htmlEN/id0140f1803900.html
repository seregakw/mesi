PCM REPROGRAMMING [MZR-CD (RF Turbo)]

id0140f1803900

Caution
•  When replacing any one injector with no circle mark in the calibration code with one that does have a circle mark, always replace the other three injectors with injectors having a circle mark at the same time and perform PCM reprogramming.
am3zzw00004809

1.  Connect the M-MDS to DLC-2.

am3uuw00000151

2.  After the vehicle is identified, select the following items from the M-MDS initialization screen.

•  When using the IDS (laptop PC)
1.  Select the “Module Programming”.
•  When using the PDS (Pocket PC)
1.  Select “Programming”.
2.  Select “Module Programming”.

3.  Then, select “Module Reprogramming” and “PCM” from the screen menu.

4.  Perform the PCM reprogramming following the instructions on the screen.
