INPUT/TURBINE SPEED SENSOR REMOVAL/INSTALLATION [FN4A-EL]

id0517a1801300

1.  Remove the battery cover. (See BATTERY REMOVAL/INSTALLATION [ZJ, ZY, Z6].) (See BATTERY REMOVAL/INSTALLATION [LF, L3].)

2.  Disconnect the negative battery cable.

3.  Remove the battery, battery box and battery tray. (See BATTERY REMOVAL/INSTALLATION [ZJ, ZY, Z6].) (See BATTERY REMOVAL/INSTALLATION [LF, L3].)

4.  Disconnect the input/turbine speed sensor connector.

am3zzw00001229

5.  Remove the input/turbine speed sensor.

6.  Apply ATF to a new O‐ring and install it on an input/turbine speed sensor.

7.  Install the input/turbine speed sensor using a new bolt.

Caution
•  Always use a new bolt. If the removed bolt is reused, it may cause oil leakage.
•  A sealant coating is applied to the input/turbine speed sensor installation bolt to prevent oil leakage from the installation bolt hole which passes through the component.
Tightening torque
8—11 N·m {82—112 kgf·cm, 71—97 in·lbf}

8.  Connect the input/turbine speed sensor connector.

9.  Install the battery, battery box and battery tray. (See BATTERY REMOVAL/INSTALLATION [ZJ, ZY, Z6].) (See BATTERY REMOVAL/INSTALLATION [LF, L3].)

10.  Connect the negative battery cable.

11.  Install the battery cover. (See BATTERY REMOVAL/INSTALLATION [ZJ, ZY, Z6].) (See BATTERY REMOVAL/INSTALLATION [LF, L3].)
