id0102c3834800

DTC P1145

Calculated torque error
DETECTION CONDITION
•  Base torque map - Not plausible
POSSIBLE CAUSE
•  Erratic signal from fuel pressure sensor
•  Fuel injector malfunction
•  Fuel metering control valve malfunction
•  Fuel leakage or clogging in fuel system
•  Fuel injection control improper operation
•  Turbo charging system improper operation
•  Engine malfunction (low compression or excessive friction loss)
•  Fuel limiter malfunction
•  Fuel supply pump malfunction
•  PCM malfunction
