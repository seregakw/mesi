PASSENGER JUNCTION BOX (PJB) CONFIGURATION

1.  Connect the M-MDS to the DLC-2 (16-pin).

am3zzw00001927

2.  After set up the M-MDS (including the vehicle recognition), the following can be selected from the M-MDS screen.

When using the IDS (Notebook PC):
•  "Module Programming"
When using the PDS (Pocket PC):
•  "Programming", "Module Programming"

3.  Select “Programmable Module Installation”.

4.  Select “GEM” and perform procedures according to directions on the M-MDS screen.

5.  Select “Programmable Parameters”.

6.  Select “GEM” and perform procedures according to directions on the M-MDS screen.

7.  Select “Programmable Module Installation”.

8.  Retrieve DTCs using the M-MDS and verify that there is no DTC present.

•  If any DTC is present, perform applicable DTC inspection. (See DTC TABLE[PJB].)
