1.  Remove the air cleaner cover. (See INTAKE-AIR SYSTEM REMOVAL/INSTALLATION[ZJ, ZY, Z6].)
U.K. specs.
1.  Set the air cleaner cover out of the way. (See INTAKE-AIR SYSTEM REMOVAL/INSTALLATION[ZJ, ZY, Z6].)

4.  Install the SST to the rear HO2S.
