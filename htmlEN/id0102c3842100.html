DTC P2585

Fuel additive control module warning lamp request
DETECTION CONDITION
•  The PCM monitors the PCL lamp condition using the PCL lamp operation signal that from the fuel additive control module to instrument cluster. When the fuel additive control module detects that the fuel additive amount is empty. The fuel additive control module sends the signal to instrument cluster and PCM. If the PCM received the signal, the PCM determines that the fuel additive control module requests the PCL lamp to ON.
POSSIBLE CAUSE
•  No fuel additive in fuel additive tank
•  Fuel additive system malfunction
•  Fuel additive level sensor malfunction
•  Fuel additive control module malfunction
