SPARK PLUG REMOVAL/INSTALLATION[ZJ, ZY, Z6]

id0118b1800400

Caution
•  If a spark plug that is not as specified is installed, sealing performance will be deteriorated. Install only the specified spark plug when replacing.

1.  Remove the battery cover. (See BATTERY REMOVAL/INSTALLATION[ZJ, ZY, Z6].)

2.  Disconnect the negative battery cable.

3.  Remove the air cleaner (except U.K.). (See INTAKE-AIR SYSTEM REMOVAL/INSTALLATION[ZJ, ZY, Z6].)

4.  Set the air cleaner cover out of the way (U.K. specs).

5.  Remove the air cleaner case (U.K. specs). (See INTAKE-AIR SYSTEM REMOVAL/INSTALLATION[ZJ, ZY, Z6].)

6.  Remove the ignition coil. (See IGNITION COIL REMOVAL/INSTALLATION[ZJ, ZY, Z6].)

7.  Remove the spark plugs using a plug-wrench.

8.  Install in the reverse order of removal.

Tightening torque
14.7―22.6 N·m {1.50―2.30 kgf·m, 10.9―16.6 ft·lbf}
