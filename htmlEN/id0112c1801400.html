RADIATOR REMOVAL/INSTALLATION[ZJ, ZY, Z6]

id0112c1801400

Warning
•  Remove and install all parts when the engine is cold, otherwise they can cause severe burns or serious injury.

1.  Remove the battery cover. (See BATTERY REMOVAL/INSTALLATION[ZJ, ZY, Z6].)

2.  Disconnect the negative battery cable.

3.  Remove the under cover.

4.  Drain the engine coolant. (See ENGINE COOLANT REPLACEMENT[ZJ, ZY, Z6].)

5.  Disconnect the coolant reserve hose from the radiator.

6.  Set the front wiring harnesses out of the way.

7.  Remove in the order indicated in the table.

am3zzw00002951

1
Fan control module connector
2
Cooling fan component
(See Cooling Fan Component Removal Note.)
(See Cooling Fan Component Installation Note.)
3
Radiator lower hose
4
Radiator upper hose
5
Radiator mount
6
Radiator
(See Radiator Removal Note.)
(See Radiator Installation Note.)

8.  Install in the reverse order of removal.

9.  Add engine coolant. (See ENGINE COOLANT REPLACEMENT[ZJ, ZY, Z6].)

10.  Inspect for engine coolant leakage. (See ENGINE COOLANT LEAKAGE INSPECTION[ZJ, ZY, Z6].)


Cooling Fan Component Removal Note

1.  Unlock the lower side tab A at two points by pressing the cooling fan side tab, remove the upper side tab A at two points, then remove the cooling fan component from the radiator.

am3zzw00002952


Radiator Removal Note

1.  Remove the oil cooler from the radiator with the hoses still connected by pulling the radiator tabs outward. (ATX)

am3zzw00002953

2.  Remove the condenser from the radiator with the pipes still connected, by pressing the radiator side tab to unlock the tab B on the condenser.

am3zzw00002954

3.  Remove rubber mount A from the mount installation hole.

am3zzw00002955

4.  Tilt the radiator to the engine side.

5.  Remove rubber mount B from the mount installation hole.

6.  Remove the radiator from below.


Radiator Installation Note

1.  Install the condenser to the radiator by aligning lower side tab B with the radiator side tab, install upper tab B, then install lower side tab B.

am3zzw00002956

2.  Install the oil cooler by inserting oil cooler bracket to the radiator tab on the radiator. (ATX)

am3zzw00002953

3.  Install the radiator.


Cooling Fan Component Installation Note

1.  Insert tab A to the radiator to install the cooling fan component.
