1.  Connect the M-MDS to the DLC-2.

am3zzw00001121

2.  After the vehicle is identified, select the following items from the initial screen of the M-MDS.

•  When using the IDS (notebook PC)
1.  Select “Powertrain”.
2.  Select “Service Function”.
3.  Select “FACM”.
•  When using the PDS (pocket PC)
1.  Select “All Tests and Calibrations”.
2.  Select “FACM”.

3.  Then, select “New FACM” and perform procedures according to the directions on the M-MDS screen.

4.  Retrieve DTC’s using the M-MDS, then verify that there in no DTC present.

•  If a DTC is present, perform the applicable DTC inspection. (See DTC TABLE[MZ-CD 1.6 (Y6)].)
