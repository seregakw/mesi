Boot Band (Crimp Type) Disassembly Note

1.  Insert a flathead screw driver into the boot band as shown in the figure, then press the caulked area outward to remove the boot band.

am3zzw00009649

Tie Rod Disassembly Note

1.  Fix the steering rack and remove the tie rod using the SST.

am3zzw00001728


Locknut (Adjusting Cover), Adjusting Cover Disassembly Note

1.  Remove the locknut using the SST.

am3zzw00001729

2.  Remove the adjusting cover.


Oil Seal Disassembly Note

1.  Install the SST (49 N032 319A) to the gear housing with the raised part facing up as shown in the figure.

2.  Insert the SSTs (49 F032 303, 49 B032 323) into the valve housing side.

3.  Remove the oil seal using a press.

am3zzw00001730


Mounting Rubber Disassembly Note

1.  Press the mounting rubber out from the gear housing using the SSTs and a press.
