AFTER REPAIR PROCEDURE[MZ-CD 1.6 (Y6)]

id0102c3801100

Note
•  The DTC P1000 will be stored in continuous memory if any of the OBD monitors do not complete their full diagnostic check. If DTC P1000 is stored in the PCM, perform the “OBD DRIVE MODE”. (See OBD DRIVE MODE[MZ-CD 1.6 (Y6)])

1.  Connect the M-MDS to the DLC-2.

am3zzw00000492

2.  Cycle the engine switch off, then to the ON position (Engine off).

3.  After the vehicle is identified, select the following items from the initial screen of the M-MDS.

•  When using the IDS (notebook PC)
1.  Select the “Toolbox” tab.
2.  Select “Self Test”.
3.  Select “Modules”.
4.  Select “PCM”.
5.  Select "Retrieve CMDTCs"
•  When using the PDS (pocket PC)
1.  Select “Module Tests”.
2.  Select “PCM”.
3.  Select “Self Test”.
4.  Select “Retrieve DTC”.

4.  Verify the DTC according to the directions on the M-MDS screen.

5.  Press the clear button on the DTC screen to clear the DTC.

6.  Verify that no DTCs are displayed.
