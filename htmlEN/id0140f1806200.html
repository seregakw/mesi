Note
•  Perform the following test only when directed.

Resistance Inspection

1.  Remove the IAT sensor No.2. (See INTAKE-AIR SYSTEM REMOVAL/INSTALLATION [MZR-CD (RF Turbo)].)

2.  Place the IAT sensor No.2 in water with a thermometer, and heat the water gradually.

3.  Measure the resistance between the IAT sensor No.2 terminals using an ohmmeter.

•  If not as specified, replace the IAT sensor No.2.
•  If IAT sensor N0.2 is okay, perform the “Circuit Open/Short Inspection”.

Circuit Open/Short Inspection

1.  Disconnect the PCM connector. (See PCM REMOVAL/INSTALLATION [MZR-CD (RF Turbo)].)

2.  Inspect for open/short circuit in the following wiring harnesses.

•  If there is open/short circuit, repair or replace wiring harnesses.

am3zzw00004085

Open circuit

•  IAT sensor No.2 terminal A and PCM terminal 79
•  IAT sensor No.2 terminal B and PCM terminal 74


Short circuit

•  IAT sensor No.2 terminal A and power supply
•  IAT sensor No.2 terminal A and body ground
•  IAT sensor No.2 terminal B and power supply
