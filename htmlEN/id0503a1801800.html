NO.13 NO KICKDOWN[FN4A-EL]

id0503a1801800

13
No kickdown
DESCRIPTION
•  Does not downshift when accelerator pedal is fully depressed within kickdown range.
POSSIBLE CAUSE
•  If transaxle does not downshift though shifting is normal, the malfunction is in the TP sensor or APPsensor circuit (including sensor GND, sensor wiring harness and connector).
