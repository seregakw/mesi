Caution
•  To prevent the A/F sensor damage, do not perform the following operation before starting the engine for the first time after the PCM is replaced.
―  Fully depress and release the accelerator pedal three times repeatedly while depressing the brake pedal with the ignition switch in the ON position.
Note
•  If the PCM is replaced, always perform each adjustment procedure using the following steps.

Set bolt removal note

1.  Using a chisel and a hammer, cut a groove on the head of the set bolt so that a screwdriver can be inserted.

2.  Loose the set bolt using an impact screwdriver or pliers.
