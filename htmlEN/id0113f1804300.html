Note
•  Perform the following test only when directed.

1.  Remove the glow plug lead. (See GLOW PLUG REMOVAL/INSTALLATION [MZR-CD (RF Turbo)].)

2.  Measure the resistance between the glow plug terminal and the cylinder head (body ground) using an ohmmeter.

•  If not as specified, replace the glow plug. (See GLOW PLUG REMOVAL/INSTALLATION [MZR-CD (RF Turbo)].)
