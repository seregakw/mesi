ENGINE COOLANT TEMPERATURE (ECT) SENSOR REMOVAL/INSTALLATION [LF, L3]

id0140a9801900

1.  Remove the battery cover.

2.  Disconnect the negative battery cable.

3.  Remove the plug hole plate. (See PLUG HOLE PLATE REMOVAL/INSTALLATION [LF, L3].)

4.  Drain the engine coolant. (See ENGINE COOLANT REPLACEMENT [LF, L3].)

5.  Disconnect the ECT sensor connector.

6.  Remove the ECT sensor using the SST.

am3zzw00001746

7.  Install in the reverse order of removal.

Tightening torque
10—14 N·m {102—142 kgf·cm, 89—123 in·lbf}
