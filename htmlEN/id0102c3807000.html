DTC P0340

Camshaft position (CMP) sensor circuit malfunction
DETECTION CONDITION
•  The PCM monitors the input voltage from the CKP sensor signal. If the voltages at PCM deviates from the threshold, the PCM determines that the there is a CKP sensor problem.
POSSIBLE CAUSE
•  CMP sensor malfunction
•  Connector or terminal malfunction
•  Short to power supply between CMP sensor terminal C and PCM terminal 206
•  Short to power supply between CMP sensor terminal B and PCM terminal 240
•  Short to power supply between CMP sensor terminal A and PCM terminal 237
•  Short to ground between CMP sensor terminal C and PCM terminal 206
•  Short to ground between CMP sensor terminal B and PCM terminal 240
•  Open circuit between CMP sensor terminal C and PCM terminal 206
•  Open circuit between CMP sensor terminal B and PCM terminal 240
•  Open circuit between CMP sensor terminal A and PCM terminal 237
•  CMP sensor pulse wheel malfunction
•  PCM malfunction
