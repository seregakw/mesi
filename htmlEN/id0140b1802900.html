KNOCK SENSOR (KS) REMOVAL/INSTALLATION[ZJ, ZY, Z6]

id0140b1802900

1.  Remove the battery cover. (See BATTERY RECHARGING[ZJ, ZY, Z6].)

2.  Disconnect the negative battery cable.

3.  Remove the intake manifold. (See INTAKE-AIR SYSTEM REMOVAL/INSTALLATION[ZJ, ZY, Z6].)

4.  Install the SST to the KS.

am3zzw00002530

5.  Remove the KS.

6.  Install in the reverse order of removal.

KS tightening torque
19.6—34.3 N·m {2.0—3.4 kgf·m, 14.5—24.5 ft·lbf}
