PCM Security Bracket And Security Bracket Sheer Bolt Removal Note

Caution
•  Do not place excessive strain on the PCM wiring harness.

1.  File flat across the PCM security bracket welded nut.

am3zzw00001032

2.  Drill a 2 mm {0.08 in} pilot hole 3 mm {0.12 in} from the outer edge of the security bracket welded nut at an angle of approximately 2°.

3.  Drill a 5 mm {0.2 in} hole in the security bracket welded nut and remove the security bracket sheer bolt.

am3zzw00001033

4.  Slide the security bracket off of the PCM.

PCM Removal/Installation Note

1.  When replacing with a new PCM, perform the following procedure:

(1)  Remove the old PCM.
(2)  Install the new PCM.
(3)  Perform the PCM parameter reset for the immobilizer system on the new PCM using the M-MDS. (See IMMOBILIZER SYSTEM COMPONENT REPLACEMENT/KEY ADDITION AND CLEARING[KEYLESS ENTRY SYSTEM].) (See IMMOBILIZER SYSTEM COMPONENT REPLACEMENT/KEY ADDITION AND CLEARING[ADVANCED KEYLESS SYSTEM].)
(4)  Remove the new PCM.
(5)  Install the old PCM.
(6)  Perform the PCM configuration using the M-MDS. (See PCM CONFIGURATION[MZ-CD 1.6 (Y6)].)
(7)  Reset the EGR valve base value using the M-MDS. (See EGR VALVE BASE VALUE RESET[MZ-CD 1.6 (Y6)].)
(8)  Reset the intake throttle valve base value using the M-MDS. (See INTAKE THROTTLE VALVE BASE VALUE RESET[MZ-CD 1.6 (Y6)].)


PCM Security Bracket And Security Bracket Sheer Bolt Installation Note
