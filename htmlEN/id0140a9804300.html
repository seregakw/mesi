MANIFOLD ABSOLUTE PRESSURE (MAP) SENSOR REMOVAL/INSTALLATION [LF, L3]

id0140a9804300

1.  Remove the battery cover.

2.  Disconnect the negative battery cable.

3.  Remove the plug hole plate. (See PLUG HOLE PLATE REMOVAL/INSTALLATION [LF, L3].)

4.  Remove the vacuum hose.

5.  Disconnect the MAP sensor connector.

6.  Remove the MAP sensor installation screw.

am3zzw00002369

7.  Remove MAP sensor from the intake manifold.

8.  Install in the reverse order of removal.

Tightening torque
2.7—3.7 N·m {28—37 kgf·cm, 24—32 in·lbf}
