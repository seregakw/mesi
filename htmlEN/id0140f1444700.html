Using M-MDS

1.  Connect the M-MDS to the DLC-2.

am3zzw00013269

2.  After the vehicle is identified, select the following items from the initial screen of the M-MDS.

When using the IDS (laptop PC)
•  Select the “Toolbox” tab.
•  Select the “Powertrain”.
•  Select “Data Reset”.
When using the PDS (Pocket PC)
•  Select “All Tests and Calibrations”.
•  Select “Data Reset”.

3.  Then, select the “Engine Oil” and perform the data reset according to directions on the M-MDS screen.

4.  Retrieve DTCs using the M-MDS, then verify if DTCs are present.

•  If a DTC is present, perform the applicable DTC inspection.


Without using M-MDS

1.  Switch the ignition to ON.

2.  Ground the test terminal using a jumper wire.

am3zzw00013276

3.  Repeatedly depress and release the accelerator pedal fully five times.

Note
•  When the engine oil data reset is completed, the glow indicator light flashes 5 times.
