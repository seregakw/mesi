DTC P1923

Fuel additive level circuit range/performance
DETECTION CONDITION
•  The PCM monitors the fuel additive level using the fuel additive control module. If the fuel additive control module detects that following items are below or above specified. The PCM determines that the fuel additive level circuit is out of range or performance problem.
―  Additive dosing status
―  Maximum additive quantity
POSSIBLE CAUSE
•  Fuel filler cap switch malfunction
•  Fuel additive supply pump malfunction
•  Fuel gauge sender unit malfunction
•  Fuel additive level sensor malfunction
•  Erratic fuel tank level signal from instrument cluster to fuel additive control module
•  Erratic required fuel additive amount signal from PCM to fuel additive control module
•  Fuel additive control module malfunction
•  PCM malfunction
•  Instrument cluster malfunction
