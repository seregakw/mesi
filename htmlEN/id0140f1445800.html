INTAKE SHUTTER VALVE POSITION SENSOR INSPECTION [MZR-CD (RF Turbo)]

id0140f1445800

Note
•  Perform the following test only when directed.

Voltage Inspection

1.  Turn the engine switch to ON.

2.  Monitor the TP1 PID using the M-MDS.

•  If not as specified, perform the “Circuit Open/Short Inspection”.
―  If there is no open/short circuit, replace the intake shutter valve. (See INTAKE-AIR SYSTEM REMOVAL/INSTALLATION [MZR-CD (RF Turbo)].)
TP1 PID
Approx. 4.2 V

3.  Start the engine and warm up the engine completely.

4.  Monitor the TP1 PID using the M-MDS at idle.

•  If not as specified, perform the “Circuit Open/Short Inspection”.
―  If there is no open/short circuit, replace the intake shutter valve. (See INTAKE-AIR SYSTEM REMOVAL/INSTALLATION [MZR-CD (RF Turbo)].)
TP1 PID
0.75—3.79 V


Circuit Open/Short Inspection

1.  Disconnect the PCM connector. (See PCM REMOVAL/INSTALLATION [MZR-CD (RF Turbo)].)

2.  Disconnect the intake shutter valve connector.

3.  Inspect for open/short circuit in the following wiring harnesses.

•  If there is open/short circuit, repair or replace wiring harnesses.
am3zzw00004210


Open circuit

•  Intake shutter valve terminal C and PCM terminal 70
•  Intake shutter valve terminal E and PCM terminal 45
•  Intake shutter valve terminal F and PCM terminal 86


Short circuit

•  Intake shutter valve terminal C and power supply
•  Intake shutter valve terminal E and power supply
•  Intake shutter valve terminal E and body ground
•  Intake shutter valve terminal F and power supply
•  Intake shutter valve terminal F and body ground
