DIESEL PARTICULATE FILTER DIFFERENTIAL PRESSURE SENSOR REMOVAL/INSTALLATION[MZ-CD 1.6 (Y6)]

id0140b2805800

1.  Remove the battery cover. (See BATTERY REMOVAL/INSTALLATION[MZ-CD 1.6 (Y6)].)

2.  Disconnect the negative battery cable.

3.  Disconnect the Diesel particulate filter differential pressure sensor connector.

4.  Remove the Diesel particulate filter differential pressure sensor installation nuts.

5.  Remove the Diesel particulate filter differential pressure sensor.

am3zzw00001595

6.  Install in the reverse order of removal.

Tightening torque
3.3—4.3 N·m {33.7—43.8 kgf·cm, 29.3—38.0 in·lbf}
