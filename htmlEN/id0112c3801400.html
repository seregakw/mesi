RADIATOR REMOVAL/INSTALLATION[MZ-CD 1.6 (Y6)]

id0112c3801400

Warning
•  Remove and install all parts when the engine is cold, otherwise they can cause severe burns or serious injury.

1.  Remove the battery cover. (See BATTERY REMOVAL/INSTALLATION[MZ-CD 1.6 (Y6)].)

2.  Disconnect the negative battery cable.

3.  Remove the engine cover.

4.  Remove the air cleaner and hose. (See INTAKE-AIR SYSTEM REMOVAL/INSTALLATION[MZ-CD 1.6 (Y6)].)

5.  Remove the air pipe and hose as a single unit. (See INTAKE-AIR SYSTEM REMOVAL/INSTALLATION[MZ-CD 1.6 (Y6)].)

6.  Remove the under cover.

7.  Drain the engine coolant. (See ENGINE COOLANT REPLACEMENT[MZ-CD 1.6 (Y6)].)

8.  Disconnect the coolant reserve hose from the radiator.

9.  Set the front wiring harness out of the way.

10.  Remove in the order indicated in the table.

11.  Install in the reverse order of removal.

12.  Add engine coolant.(See ENGINE COOLANT REPLACEMENT[MZ-CD 1.6 (Y6)].)

13.  Inspect for engine coolant leakage. (See ENGINE COOLANT LEAKAGE INSPECTION[MZ-CD 1.6 (Y6)].)


Cooling Fan Component Removal Note

1.  Unlock the lower side tab A at two points by pressing the cooling fan side tab, remove the upper side tab A at two points, then remove the cooling fan component from the radiator.

am3zzw00001429


Radiator Removal Note

1.  Remove the condenser from the radiator with the pipes still connected, by pressing the radiator side tab to unlock the tab B on the condenser.

am3zzw00001430

2.  Remove rubber mount A from the mount installation hole.

am3zzw00001431

3.  Tilt the radiator to the engine side.

4.  Remove rubber mount B from the mount installation hole.

5.  Remove the radiator from below.


Radiator Installation Note

1.  Install the condenser to the radiator by aligning lower side tab B with the radiator side tab, install upper tab B, then install lower side tab B.

am3zzw00001430

2.  Install the radiator.


Cooling Fan Component Installation Note

1.  Insert tab A to the radiator to install the cooling fan component.
