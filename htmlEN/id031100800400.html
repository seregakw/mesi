Locknut (Crimped-fix Type) Removal Note

Note
•  The locknuts include two types; a self-lock type and a crimped-fix type. Verify which type of locknut is used on the vehicle being serviced. If it is a crimped-fix type locknut, remove it using the following procedure.

1.  Knock the crimped portion of the locknut outward using a small chisel and a hummer.

am3zzw00004361

2.  Lock the hub by applying the brakes.

3.  Remove the locknut.


Front Lower Arm Ball Joint Removal Note

1.  Loosen the installation bolt (front side).

am3zzw00004362

2.  Loosen the installation bolts (rear side).

3.  Disconnect the front lower arm ball joint.


Wheel Hub Component Removal Note

1.  Remove the wheel hub component using the SSTs.

am3zzw00004547

2.  If the bearing inner race remains on the front wheel hub component, perform the following procedure:


Without using SST

1.  Grind a section of the bearing inner race until approx. 0.5 mm {0.02 in} remains.

am3uuw00001203

2.  Remove the bearing inner race using a chisel.


With using SST

1.  Remove the part as shown in the figure.

am3uuw00001204

2.  Position the SSTs and a spare bolt (M10 length 90 mm {3.5 in} or less) as shown in the figure.

am3uuw00001205

Caution
•  When tightening section A, tighten section B securely because the engagement of the SST tab is shallow and it can come off easily.

3.  Tighten SST section A until the space between the bearing inner race and wheel hub component is 1—2 mm {0.04—0.07 in}.

am3zzw00004550

4.  Temporarily loosen SST section A and position the SST again.

5.  Tighten SST section A and remove the bearing inner race.

Caution
•  Install the wheel bearing with the ABS encoder (tea color) facing inside of the vehicle.

Wheel Hub Component Installation Note

1.  Install the wheel hub component so that there is no clearance between the wheel hub and the bearing using the SSTs.

Front Lower Arm Ball Joint Installation Note

1.  Tighten the installation bolts.

Tightening torque
•  Front side: 130.0—150.0 N·m {13.26—15.29 kgf·m, 95.9—110.5 ft·lbf}
•  Rear side: 75.5—102.0 N·m {7.70—10.4 kgf·m, 55.7—75.2 ft·lbf}
