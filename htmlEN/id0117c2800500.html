Battery Cover Removal Note

Caution
•  To prevent damage to the clips, remove the battery cover one side at a time.
•  When pulling the clips, do not apply excessive force to a clip using the screwdriver or pair of pliers.
am3zzw00007293

1.  Remove the battery cover using the following procedure.

am3zzw00007294

(1)  Pull the clip (LH) in the outward direction and disengage the battery cover tabs.
(2)  Pull the clip (RH) in the outward direction and disengage the battery cover tabs.
(3)  Remove the battery cover.


Set Bolt Removal Note (U.K. Specs.)

1.  Drill the set bolts using a drill bit with a diameter larger than the shanks until the heads are removed.

•  If the shanks are remained, remove them using locking pliers.

2.  Clean all foreign material from the PCM connectors.


PCM Cover Installation Note (Except for U.K. Specs.)

1.  Install with the PCM cover hooks B aligned with the PCM box holes.

am3zzw00001141

2.  Install the PCM cover to the PCM box hook.

am3zzw00001142


PCM Cover Installation Note (U.K. Specs)

1.  Insert the tab of the PCM cover to the hole.

am3zzw00001143

2.  Rotate until area F is parallel. The harness and the PCM cover should be located at area G.

am3zzw00001144


Set Bolt Installation Note

1.  Install new set bolts and temporarily tighten them first, then tighten them until the necks of the bolts break off.


Battery Box Installation Note

1.  Assemble with battery box hooks E aligned with the battery tray holes at two points.

am3zzw00001145


Battery Duct Installation Note

1.  Match the mark of the shroud panel and notch in the battery duct, and install the battery duct to the shroud panel.

am3zzw00001146


Battery Cover Installation Note

1.  Install the battery duct between the battery cover and the battery box.

am3zzw00001147
