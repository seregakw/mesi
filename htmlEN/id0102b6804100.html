DTC P0118

ECT sensor circuit high input
DETECTION CONDITION
•  The PCM monitors the ECT sensor signal. If the PCM detects the ECT sensor voltage is above 4.58 V, the PCM determines that the ECT sensor circuit has malfunction.
Diagnostic support note
•  This is a continuous monitor (Engine cooling system).
•  The MIL illuminates if the PCM detects the above malfunction condition during the first drive cycle.
•  FREEZE FRAME DATA is available.
•  The DTC is stored in the PCM memory.
POSSIBLE CAUSE
•  ECT sensor malfunction
•  Connect or terminal malfunction
•  Open circuit in wiring harness between ECT sensor terminal A and PCM terminal 2AK
•  Short to power supply in wiring harness between ECT sensor terminal A and PCM terminal 2AK
•  Open circuit in wiring harness between ECT sensor terminal B and PCM terminal 2AA
•  PCM malfunction
