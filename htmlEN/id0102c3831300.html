DTC P0344

Camshaft position (CMP) sensor circuit intermittent concern
DETECTION CONDITION
•  The PCM monitors the input voltage from the CMP sensor signal. If the voltages at PCM deviates from the threshold, the PCM determines that the there is a CMP sensor problem.
POSSIBLE CAUSE
•  CMP sensor circuit intermittent concern
•  CMP sensor pulse wheel malfunction
•  PCM malfunction
