BRAKE FLUID PRESSURE SENSOR INITIALIZATION PROCEDURE

id041500802200

Warning
•  If the initialization procedure is not completed, the DSC will not operate properly and it might cause an unexpected accident. Therefore, after replacing or removing the DSC HU/CM, make sure to perform the initialization procedure to insure proper DSC operation.

1.  Turn the ignition switch off.

2.  Connect the M-MDS to the DLC-2.

am3zzw00001649

3.  After the vehicle is identified, select the following items from the initial screen of the M-MDS.

•  When using the IDS (laptop PC)
1.  Select “Chassis“.
2.  Select “ABS/DSC“.
3.  Select “Sensor Initialization“
•  When using the PDS (Pocket PC)
1.  Select “Module Tests”.
2.  Select “ABS”.
3.  Select “DataLogger”.
4.  Select “MCYL_S_CAL”.

4.  Perform the initialization procedure according to the directions on the screen.

5.  Drive the vehicle.

6.  After 5 min or more of driving, verify that the DSC system is normal.
