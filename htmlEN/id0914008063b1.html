KEYLESS CONTROL MODULE REMOVAL/INSTALLATION[ADVANCED KEYLESS SYSTEM]

Note
•  If the keyless control module is replaced, always perform the following procedure.
Without immobilizer system
―  Configuration (See KEYLESS CONTROL MODULE CONFIGURATION[ADVANCED KEYLESS SYSTEM])
―  Card key programming (See CARD KEY ID CODE REGISTRATION[ADVANCED KEYLESS SYSTEM].)
―  Steering lock unit programming (See STEERING LOCK UNIT ID CODE REGISTRATION[ADVANCED KEYLESS SYSTEM].)
With immobilizer system
―  Configuration (See KEYLESS CONTROL MODULE CONFIGURATION[ADVANCED KEYLESS SYSTEM])
―  Card key programming (See CARD KEY ID CODE REGISTRATION[ADVANCED KEYLESS SYSTEM].)
―  Steering lock unit programming (See STEERING LOCK UNIT ID CODE REGISTRATION[ADVANCED KEYLESS SYSTEM].)
―  Immobilizer system resetting (with immobilizer system) (See IMMOBILIZER SYSTEM COMPONENT REPLACEMENT/KEY ADDITION AND CLEARING[ADVANCED KEYLESS SYSTEM].)

1.  Disconnect the negative battery cable.

2.  Remove the glove compartment. (See GLOVE COMPARTMENT REMOVAL/INSTALLATION.)

3.  Remove the PJB. (See PASSENGER JUNCTION BOX (PJB) REMOVAL/INSTALLATION.)

4.  Disconnect the keyless control module connector.

5.  Remove the nuts, then remove the keyless control module.

6.  Install in the reverse order of removal.
